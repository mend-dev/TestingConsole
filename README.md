# Commands
- bloon
  - spawn (ID) (AMOUNT) (SPACING)
  - clear
  - clearall

- tower
  - spawn (ID) (XPOS) (YPOS)
  - clear
  - clearall
  - sell
  - sellall

- cash
  - set (AMOUNT)
  - add (AMOUNT)
  - remove (AMOUNT)

- lives
  - set (AMOUNT)
  - add (AMOUNT)
  - remove (AMOUNT)

- round
  - set (AMOUNT)
  - add (AMOUNT)
  - remove (AMOUNT)

## Examples:
To spawn ddts: bloon spawn ddt 5 10
To set cash: cash set 500000
To spawn tower: tower spawn dart 0 0
